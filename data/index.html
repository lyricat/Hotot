    Rule Editor

    - Rule Name -

    - Rule -

    Examples

    • ABC, drop tweets contain word "ABC".
    • via:foursquare, drop tweets sent from foursquare.
    • name:MentionTo, drop tweets sent by @MentionTo.
    • tag:nowplaying, drop tweets contain hash tag #nowplaying.
    • mention:hotot do:notifynotify me if incoming tweets mention @hotot
    • #GossipGirl do:mask, mask tweets contain hash tag #GossipGirl.

    Quick References

    • keyword

      simply a keyword

    • via:{source name}

      where this tweet comes form.

      example: via:foursquare

    • tag:{tag name}

      What hash tag contained by this tweet.

      example: via:#FuckGFW

    • name:{screen name}

      who post this tweet.

      example: name:MentionTo

    • mention:{screen name}

      who are mentioned in this tweet.

      example: mention:aiww

    • has:{attribute}

      what attribute this tweet has.

      supported attributes: has:geo, has:url

    • do:{action}

      take action if all conditions are true.

      supported action:

      • do:drop, drop this tweet.
      • do:mask, mask this tweet.
      • do:notify, notify this tweet.
      • do:archive, archive this tweet.
      • do:reply({text}), reply this tweet.
      • do:quote({text}), quote this tweet.
